# Cobain

Clever, target oriented, focus on EVM, memcoin and SOMNIA

## Features

- Staking
- Access Control
- Price Monitoring
- Automated Trading

## Quick Start

```bash
# Install dependencies
forge install foundry-rs/forge-std

# Run tests
forge test

# Deploy
forge script script/Deploy.s.sol --rpc-url somnia_testnet --broadcast
```
